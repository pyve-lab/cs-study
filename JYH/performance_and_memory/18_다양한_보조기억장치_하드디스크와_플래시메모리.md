# 다양한 보조기억장치: 하드디스크와 플래시 메모리

전원이 꺼져도 저장된 내용을 유지하는 **비휘발성** 저장 장치인 보조기억장치의 대표 주자, 하드디스크(HDD)와 플래시 메모리의 구조와 동작 원리를 정리합니다.

---

## 1. 하드디스크 (Hard Disk Drive, HDD)
자기적인 방식으로 데이터를 저장하는 '자기 디스크'의 일종입니다.

### 1) 주요 구성 요소
* **플래터(Platter)**: 데이터가 저장되는 동그란 원판입니다. 자기 물질이 입혀져 있어 n극과 s극으로 데이터를 표현합니다.
* **스핀들(Spindle)**: 플래터를 회전시키는 축입니다. 회전 속도는 **RPM**(1분당 회전수) 단위를 사용합니다.
* **헤드(Head)**: 플래터 위를 미세하게 떠다니며 데이터를 읽고 쓰는 장치입니다.
* **디스크 암(Disk Arm)**: 헤드를 원하는 위치로 이동시키는 팔 역할을 합니다.



### 2) 데이터 저장 단위
* **트랙(Track)**: 플래터의 동심원 형태 저장 단위입니다.
* **섹터(Sector)**: 트랙을 여러 조각으로 나눈 최소 저장 단위입니다. (보통 512바이트~4KB)
* **실린더(Cylinder)**: 여러 장의 플래터에서 같은 위치에 있는 트랙들을 논리적으로 묶은 원통 모양의 단위입니다. 헤드를 움직이지 않고 한 번에 읽을 수 있는 단위이기도 합니다.

### 3) 데이터 접근 시간
1. **탐색 시간(Seek Time)**: 헤드를 해당 트랙까지 이동시키는 시간입니다.
2. **회전 지연(Rotational Latency)**: 헤드 아래에 원하는 섹터가 올 때까지 플래터를 돌리는 시간입니다.
3. **전송 시간(Transfer Time)**: 하드디스크와 컴퓨터 간에 데이터를 실제로 주고받는 시간입니다.

---

## 2. 플래시 메모리 (Flash Memory)
전기적으로 데이터를 읽고 쓰는 반도체 기반 저장 장치입니다. SSD, USB, SD 카드 등에 사용됩니다.

### 1) 셀(Cell) 타입에 따른 분류
한 셀에 몇 비트를 저장하느냐에 따라 성능, 가격, 수명이 달라집니다.

| 종류 | 셀당 비트 | 특징 |
| :--- | :---: | :--- |
| **SLC** (Single) | 1비트 | 가장 빠르고 수명이 길지만, 가격이 매우 비쌈 |
| **MLC** (Multi) | 2비트 | SLC보다 느리지만 대용량화 유리, 적절한 가격 |
| **TLC** (Triple) | 3비트 | 용량 대비 가격이 가장 저렴하나 속도와 수명이 낮음 |



### 2) 플래시 메모리의 읽기/쓰기/삭제 단위
플래시 메모리는 **읽기/쓰기 단위와 삭제 단위가 다릅니다.**
* **읽기 및 쓰기**: **페이지(Page)** 단위로 수행됩니다.
* **삭제**: **블록(Block)** 단위(페이지의 묶음)로 수행됩니다.

### 3) 가비지 컬렉션 (Garbage Collection)
플래시 메모리는 기존 데이터 위에 **덮어쓰기가 불가능**합니다.
* **과정**: 
    1. 수정된 데이터는 새로운 페이지(Free -> Valid)에 기록합니다.
    2. 기존 데이터는 '유효하지 않음(Invalid)' 상태가 됩니다.
    3. **가비지 컬렉션**을 통해 유효한 페이지만 새 블록으로 복사하고, 기존 블록을 통째로 삭제하여 빈 공간을 확보합니다.



---

## 3. 요약
* **HDD**는 물리적으로 원판을 돌리는 '기계식' 장치이므로 속도에 한계가 있습니다.
* **플래시 메모리**는 '반도체' 방식이며, 덮어쓰기가 안 되기 때문에 **가비지 컬렉션**이라는 독특한 관리 방식이 필요합니다.
