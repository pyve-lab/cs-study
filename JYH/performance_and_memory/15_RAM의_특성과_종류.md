# RAM의 특성과 종류

메모리의 핵심인 RAM의 특성과 다양한 종류에 대해 정리합니다.

---

## 1. RAM의 특징: 휘발성 저장 장치

* **휘발성(Volatile)**: 전원이 꺼지면 저장된 내용이 모두 사라지는 특성입니다.
* **보조기억장치와의 차이**: SSD, HDD, USB 등은 전원이 꺼져도 내용이 유지되는 **비휘발성** 저장 장치입니다.
* **역할**: 보조기억장치에는 '보관할 대상'을, RAM에는 '실행할 대상'을 저장합니다.

---

## 2. RAM의 용량과 성능

* **용량과 속도**: RAM 용량이 크면 많은 데이터를 미리 보조기억장치에서 가져와 저장할 수 있습니다.
* **성능 향상**: CPU가 보조기억장치에 접근하는 빈도를 줄여주기 때문에, 많은 프로그램을 동시에 실행(멀티태스킹)할 때 성능이 체감됩니다.
* **비유**: 책상(RAM)이 넓을수록 여러 권의 책(프로그램)을 펼쳐놓고 작업하기 수월한 것과 같습니다.

---

## 3. RAM의 종류

### ① DRAM (Dynamic RAM)
* **특징**: 데이터가 시간이 지나면 동적으로 사라지는 램입니다.
* **재활성화(Refresh)**: 데이터 소멸을 막기 위해 주기적으로 다시 저장(리프레시)해야 합니다.
* **장점**: 집적도가 높아 대용량으로 만들기 쉽고 가격이 저렴합니다.
* **용도**: 우리가 흔히 말하는 **메인 메모리(주기억장치)**로 사용됩니다.

### ② SRAM (Static RAM)
* **특징**: 전원이 공급되는 동안 데이터가 사라지지 않는 정적인 램입니다.
* **장점**: DRAM보다 속도가 매우 빠릅니다.
* **단점**: 집적도가 낮고 가격이 비싸며 소비 전력이 큽니다.
* **용도**: 대용량일 필요는 없지만 속도가 중요인 **캐시 메모리**로 사용됩니다.

| 구분 | DRAM | SRAM |
| :--- | :--- | :--- |
| **재충전(Refresh)** | 필요함 | 필요 없음 |
| **속도** | 느림 | 빠름 |
| **가격** | 저렴함 | 비쌈 |
| **집적도(용량)** | 높음 | 낮음 |
| **주요 용도** | 주기억장치(RAM) | 캐시 메모리 |

---

## 4. 발전된 형태의 RAM

### SDRAM (Synchronous DRAM)
* CPU의 **클럭 신호와 동기화**되어 박자에 맞춰 데이터를 주고받는 발전된 DRAM입니다.

### DDR SDRAM (Double Data Rate SDRAM)
* 가장 대중적으로 사용되는 RAM입니다.
* **대역폭(Bandwidth)**을 넓혀 데이터를 주고받는 길의 너비를 확장한 방식입니다.
* **발전 단계**: 숫자가 올라갈수록 대역폭이 2배씩 증가합니다.
    * **SDR**: 1차선 도로 (한 번에 하나 전송)
    * **DDR**: 2차선 도로 (SDR의 2배)
    * **DDR4**: 16차선 도로 (SDR의 16배) - 현재 가장 많이 쓰이는 규격
