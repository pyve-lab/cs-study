
## 1. 컴퓨터가 이해하는 정보

컴퓨터는 크게 두 가지 형태의 정보를 처리합니다.

* **데이터 (Data):** 숫자, 문자, 이미지, 동영상 같은 정적인 정보를 의미합니다.
    * *예시:* `1`, `2`, `안녕하세요`

* **명령어 (Instruction):** 데이터를 움직이고 컴퓨터를 실질적으로 동작시키는 정보입니다.
    * *예시:* `더하라`, `출력하라`
    * *관계:* 데이터는 명령어를 수행하기 위한 일종의 **재료**입니다.

## 2. 컴퓨터의 4가지 핵심 부품

컴퓨터의 종류 (라즈베리파이, 스마트폰, 노트북, 서버 등) 와 관계없이 컴퓨터를 컴퓨터답게 만들고
동작시키는 핵심부품은 거의 동일합니다.
핵심 부품의 종류에는 CPU, 메모리, 보조 기억장치, 입출력장치가 있습니다.

### ① CPU (중앙처리장치)
메모리에 저장된 명령어를 읽고, 해석하며 실행하는 컴퓨터의 두뇌입니다.

* **ALU (산술논리연산장치):** 계산기 역할 (산술 및 논리 연산 수행)을 합니다.

* **제어장치 (Control Unit):** 제어 신호(전기 신호)를 내보내고 명령어를 해석합니다.

* **레지스터 (Register):** CPU 내부의 아주 빠른 임시 저장 장치입니다.

> **CPU 동작 흐름 (시뮬레이션):**
> 1. 제어장치가 메모리에 '읽기' 신호를 보냅니다.
> 2. 메모리 특정 주소의 명령어를 레지스터로 가져옵니다.
> 3. 제어장치가 명령어를 해석합니다.
> 4. ALU가 계산 후 결과를 레지스터나 메모리에 저장합니다.

### ② 메모리 (주기억장치, RAM)
현재 실행되는 프로그램의 명령어와 데이터를 저장하는 부품입니다.

* **특징:** 전원이 꺼지면 내용이 사라지는 **휘발성** 장치입니다. 메모리 용량이 크면(예: 32기가) 담을 수 있는 명령어와 데이터의 종류가 매우 많아지므로, 원하는 정보의 위치를 측정하기 위해 주소 개념을 사용합니다.

* **주소 (Address):** 원하는 정보가 어디에 있는지 찾기 위해 위치마다 번호를 매긴 개념입니다.

* **참고:** 보통 RAM을 메모리라고 부르며, ROM도 존재하지만 학습 효율상 RAM을 위주로 다룹니다.

### ③ 보조기억장치 (HDD, SSD, USB 등)
전원이 꺼져도 저장된 내용을 보관할 수 있는 저장 장치 입니다(비휘발성).
* **역할:** '보관할' 프로그램과 데이터를 저장합니다.
* **메모리와의 차이:** 메모리는 '실행 중'인 것을 담고, 보조기억장치는 '보관 중'인 것을 담습니다.

### ④ 입출력장치 (마이크, 키보드, 모니터 등)
컴퓨터 외부와 내부가 정보를 주고받을 수 있도록 연결하는 부품입니다.
* **주변장치:** 보조기억장치와 입출력장치를 통칭하기도 합니다. (보조기억장치는 메모리를 돕는 특별한 기능을 수행하는 입출력장치의 일종으로 볼 수 있습니다.)

## 3. 메인보드와 시스템 버스

부품들이 서로 연결되어 정보를 주고받습니다.

* **메인보드 (마더보드):** 핵심 부품들을 부착하는 판이라고 볼 수 있습니다.

* **시스템 버스 (System Bus):** 부품 간의 통로이며 세 가지 길로 나뉩니다.
    1.  **주소 버스 (Address Bus):** 주소를 주고받습니다.
    2.  **데이터 버스 (Data Bus):** 명령어와 데이터를 주고받습니다..
    3.  **제어 버스 (Control Bus):** 제어 신호를 주고받습니다..

---
**요약:** 컴퓨터는 **시스템 버스**라는 통로 위에서 **CPU, 메모리, 보조기억장치, 입출력장치**가 유기적으로 연결되어 **명령어와 데이터**를 처리하는 시스템입니다.
