# RAM의 특징과 종류

## 1️⃣ RAM이란?

- **주기억장치(Main Memory)**의 한 종류
- CPU가 **직접 접근**하여 데이터를 읽고 쓰는 메모리
- 실행 중인 프로그램과 데이터를 저장

---

## 2️⃣ RAM의 공통 특징

### 🔹 휘발성 저장장치
- 전원이 꺼지면  
  → 저장된 데이터가 모두 사라짐

### 🔹 랜덤 액세스(Random Access)
- 메모리의 어떤 위치든  
  → 접근 시간은 동일

### 🔹 읽기와 쓰기 가능
- 데이터를 읽고 쓸 수 있음
- 실행 중인 프로그램의 데이터가 계속 변경됨

---

## 3️⃣ RAM의 종류

RAM은 다음과 같이 구분됨
- DRAM
- SRAM
- SDRAM
- DDR SDRAM

---

## 4️⃣ DRAM (Dynamic RAM)

### 🔹 DRAM이란?
- **가장 일반적으로 사용되는 RAM**
- 주기억장치로 사용됨

---

### 🔹 특징
- 시간이 지나면  
  → 저장된 데이터가 점차 사라짐
- 이를 방지하기 위해  
  → **주기적으로 재충전(리프레시)** 필요

---

### 🔹 장점
- 구조가 단순
- 대용량 구현에 유리
- 가격이 저렴

---

### 🔹 단점
- 재충전 과정 때문에  
  → SRAM보다 속도가 느림

---

## 5️⃣ SRAM (Static RAM)

### 🔹 SRAM이란?
- **캐시 메모리**로 사용되는 RAM
- DRAM보다 빠른 속도를 가짐

---

### 🔹 특징
- 전원이 공급되는 동안  
  → 데이터가 유지됨
- **재충전이 필요 없음**

---

### 🔹 장점
- 접근 속도가 빠름

### 🔹 단점
- 구조가 복잡
- 가격이 비쌈
- 대용량 구현에 불리

---

## 6️⃣ SDRAM (Synchronous DRAM)

### 🔹 SDRAM이란?
- **클럭 신호에 맞춰 동작하는 DRAM**
- CPU 클럭과 **동기화**되어 동작

---

### 🔹 특징
- 클럭 신호에 맞춰  
  → 데이터를 읽고 씀
- 기존 DRAM보다  
  → **성능이 향상**

---

## 7️⃣ DDR SDRAM (Double Data Rate SDRAM)

### 🔹 DDR SDRAM이란?
- SDRAM의 한 종류
- **클럭 신호의 상승과 하강**  
  → **두 번 데이터를 전송**

---

### 🔹 특징
- 같은 클럭 속도에서  
  → SDRAM보다 **전송 속도가 두 배**
- 현재 가장 널리 사용되는 RAM

---

## 8️⃣ RAM 종류 요약 비교

| 구분 | DRAM | SRAM | SDRAM | DDR SDRAM |
|---|---|---|---|---|
| 재충전 | 필요 | 불필요 | 필요 | 필요 |
| 클럭 동기화 | ❌ | ❌ | ⭕ | ⭕ |
| 데이터 전송 | 1회 | 1회 | 1회 | 2회 |
| 주 용도 | 주기억장치 | 캐시 메모리 | 주기억장치 | 주기억장치 |

---

## 9️⃣ 정리

- RAM은 CPU가 직접 접근하는 주기억장치
- DRAM
  - 저렴, 대용량
- SRAM
  - 빠름, 캐시 메모리
- SDRAM
  - 클럭과 동기화된 DRAM
- DDR SDRAM
  - 클럭당 두 번 데이터 전송

---

## ✅ 핵심 요약
- 모든 RAM은 휘발성 메모리
- 종류에 따라  
  → 속도, 구조, 용도가 다름
- 현재 주기억장치는  
  → **DDR SDRAM**이 사용됨
