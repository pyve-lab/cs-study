# 컴퓨터 구조 및 운영체제 학습 (CS Foundation)

## 학습 개요
- **기간**: 2025.12.15 ~ 2026.01.25 (Phase 2)
- **교재**: 혼자 공부하는 컴퓨터 구조 + 운영체제
- **강의**: 저자 직강 유튜브 (총 42강)

## 학습 계획

### Phase 2: 컴퓨터 구조 및 운영체제 (CS Foundation)

#### Part 1. 컴퓨터 구조 (Hardware)

**Week 6** (12.15 ~ 12.21) - CPU의 작동 원리
- 강의 시청: 약 3시간 10분
- 1강~11강: 컴퓨터 구조 기초, CPU 내부 구성, 명령어 사이클

**Week 7** (12.22 ~ 12.28) - 성능 향상과 메모리 (Lite Week)
- 강의 시청: 약 3시간 20분
- 12강~22강: CPU 설계 기법, 메모리, 보조기억장치, 입출력

**Week 8** (12.29 ~ 01.04) - Linux 서버 구축 (Mission Week)
- 실습 주간 / 강의 없음
- 개인 리눅스 서버 구축 (AWS Free Tier 또는 WSL2)
- 터미널 접속 환경 설정 (SSH)

#### Part 2. 운영체제 (Operating System)

**Week 9** (01.05 ~ 01.11) - 프로세스와 스레드 (★ 핵심)
- 강의 시청: 약 2시간 20분
- 23강~30강: 운영체제 기초, 프로세스, 스레드, 실습

**Week 10** (01.12 ~ 01.18) - 스케줄링 & 동기화
- 강의 시청: 약 2시간
- 31강~36강: CPU 스케줄링, 프로세스 동기화, 교착 상태

**Week 11** (01.19 ~ 01.25) - 메모리 & 파일시스템
- 강의 시청: 약 2시간 20분
- 37강~42강: 가상 메모리, 페이징, 파일 시스템

### Phase 3: 네트워크 & 리눅스 실무 (Linux Master)

**Week 12** (02.02 ~ 02.08) - Network Basics
- TCP/IP 4계층, IP, Port, DNS 개념 학습
- [Daily] 정처기 프로그래밍(C/Java) 1일 1문제 시작

**Week 13** (02.09 ~ 02.15) - Linux Exam Prep I
- 리눅스 일반, 리눅스 운영 및 관리 기출 풀이
- 주요 명령어 실습 (chmod, grep, vi, netstat)

**Week 14** (02.16 ~ 02.22) - Linux Exam Prep II
- 리눅스 활용(네트워크 서비스) 심화 학습
- 오답 노트 작성 및 암기 과목 정리

**Week 15** (02.23 ~ 03.01) - Final Check
- 최종 모의고사 진행 (시간 배분 연습)
- 취약 파트(네트워크 등) 보충 학습

**Week 16** (03.02 ~ 03.08) - Certification
- [D-Day] 리눅스마스터 2급 2차 시험 (예상)
