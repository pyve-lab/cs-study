## 🧩 핵심 요약
컴퓨터는 크게 아래 4가지 부품이 협력해서 동작한다.

- **CPU**: 계산/판단/제어(명령 실행의 중심)
- **메모리**: CPU가 지금 당장 처리할 데이터/명령을 올려두는 작업 공간
- **보조기억장치**: 전원이 꺼져도 남는 저장 공간(프로그램/파일 보관)
- **입출력장치**: 사용자·외부와 데이터를 주고받는 통로(입력/출력)

---

## 🧱 4가지 핵심 부품

### CPU
- 역할: **명령어를 해석하고 실행**하며, 전체 흐름을 제어
- 포인트: CPU는 “빠르게 일하지만” 저장 공간이 부족해서 **메모리와 함께** 일함

### 메모리
- 역할: CPU가 처리 중인 **데이터/명령어를 임시로 저장**
- 포인트: 실행 중인 프로그램이 사용하는 공간(일종의 작업대)

### 보조기억장치
- 역할: 프로그램/사진/문서 등 **장기 저장**
- 예시: SSD/HDD 등
- 포인트: 메모리보다 느리지만 **꺼져도 데이터가 유지**됨

### 입출력장치
- 역할: 컴퓨터 밖과 **정보를 입력/출력**
- 예시:
  - 입력: 키보드, 마우스, 마이크
  - 출력: 모니터, 스피커, 프린터
